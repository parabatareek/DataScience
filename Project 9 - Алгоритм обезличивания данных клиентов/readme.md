Разработка алгоритма обезличивания данных клиентов

# Инструменты #
1. pandas
2. numpy
3. sklearn

# Описание данных #
    Набор данных находится в файле /datasets/insurance.csv.
    Признаки:

        - количество членов его семьи
    Целевой признак:
        - количество страховых выплат клиенту за последние 5 лет.
